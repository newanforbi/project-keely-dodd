# project-keely-dodd

**Live site:** https://newanforbi.github.io/project-keely-dodd/

An interactive administrative accountability dashboard built with React and Vite, deployed via GitHub Pages.

## Development

```bash
npm install
npm run dev
```

## Build & Deploy

The site is automatically built and deployed to GitHub Pages on every push to `main`/`master` via GitHub Actions.

To deploy manually:

```bash
npm run build   # outputs to dist/
```

Make sure GitHub Pages is configured to use **GitHub Actions** as the source (repo Settings → Pages).
